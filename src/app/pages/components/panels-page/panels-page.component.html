Panels

Description

Angular4 panels custom component

Example

Setup

Add PanelModule into your current module, where you will use panel, or into ComponentsModule class.components-module.module would look like this

import { NgModule } from '@angular/core'; import { PanelModule } from '../../components/panels/panel.module'; @NgModule({ imports: [PanelModule] }) export class ComponentsModule { }

Add PanelService to your root component providers:

import { PanelService } from '../../components/panels/panel.service'; ... providers: [PanelService]

Config

In app.module.ts configurate panels:

... import { PANEL_CONFIG, PanelConfigInterface } from './components/panels/panel.config'; ... const CUSTOM_PANEL_CONFIG: PanelConfigInterface = { leftPanelExpandedShift: 250 }; ... @NgModule( { ... providers: [ ... { provide: PANEL_CONFIG, useValue: CUSTOM_PANEL_CONFIG } ... ] ... } ) ...

Available parameters:

  • [leftPanelExpandedShift] - specify shift for left panel when left sidebar is expanded ( default - 0 ).
  • [leftPanelCollapsedShift] - specify shift for left panel when left sidebar is collapsed ( default - 0 ).
  • [rightPanelExpandedShift] - specify shift for right panel when right sidebar is expanded ( default - 0 ).
  • [rightPanelCollapsedShift] - specify shift for right panel when right sidebar is collapsed ( default - 0 ).

Markup

Add this to root component:

{{'<'}}app-panels [overlay]="false">{{'<'}}/app-panels>

Parameters:

  • [overlay] - on/off page overlay when panel opened.
  • [leftPanelExpand] - specify event for subscribe to left sidebar expand/collapse.
  • [leftPanelHide] - specify event for subscribe to left sidebar show/hide.
  • [rightPanelExpand] - specify event for subscribe to right sidebar expand/collapse.
  • [rightPanelHide] - specify event for subscribe to right sidebar show/hide

And after create your panel where you need:

{{'<'}}app-panel [id]="panel-1" [dir]="'right'"> -- Your content -- {{'<'}}/app-panel>

Parameters:

  • [id] - *required

  • [dir] - direction where show panel. Posible: right, left. Default - left.

!NOTE. If root component and your component have different module imports, you must add PanelModule to both!

Open/Close panel

Just add to button appropriate parameter with panel id:

  • open panel {{'<'}}a class="btn" [openPanel]="'panel-1'">Open Panel 1{{'<'}}/a>
  • close panel {{'<'}}a class="btn" [closePanel]="'panel-1'">Open Panel 1{{'<'}}/a>

Or control panel manually from panel.service:

import {{'{'}} PanelService {{'}'}} from '~panel.service'; ... constructor(private _panelService: PanelService) {{'{'}}{{'}'}}
  • open panel this._panelService.openPanel('panel-1');
  • close panel this._panelService.closePanel('panel-1');
  • close last opened panel this._panelService.closePanel();
